Ping Host

Setting Value Value Value Value
Documentation Simple host reachability test using system 'ping' command. Verifies the return code, should be 0 if everything is OK. Run by executing: pybot ping_host.html
This file also demonstrates use of tags. To execute testcase marked with 'linux' tag:
pybot -i linux ping_host.html

Host Should Respond To Ping on Windows [Documentation] Pings the host specified in the hostname variable. Succeeds when the return code is 0.
[Tags] windows
[Timeout] 20 seconds Host ${hostname} did not respond to ping within 20 seconds
${RC} ${output} = Run and return RC and Output ping ${hostname}
Log RC: ${RC}\nOutput:\n${output}
Should be equal ${RC} ${0} Host ${hostname} unreachable. Reason:\n${output}
