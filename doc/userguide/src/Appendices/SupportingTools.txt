Supporting tools
----------------

In this appendix, the available supporting tools for Robot Framework
are described. Unless otherwise stated later in this chapter, the
tools can be downloaded from the `Robot Framework supporting tools page`_.

.. contents::
   :depth: 2 
   :local: 


Library documentation tool (libdoc.py)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`Libdoc`__ is tool for generating HTML or XML documentation from keywords of a
test library or a resource file.

__ libdoc.html


Start/End/Elapsed time reporting tool (times2csv.py)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 

`times2csv`__ is a tool for generating start, end and elapsed
time information about suites, tests and keywords in CSV format.

__ times2csv.html


Historical reporting tool (risto.py)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`Ristopy`__ is a tool for generating plots about historical statistics
of test executions.

__ ristopy.html


Tool for viewing debug files (fileviewer.py)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`Fileviewer`__ is a graphical tool implementing unix :cli:`tail` -like 
functionality. It is especially designed to view `debug files`__

__ fileviewer.html
__ `Debug file`_


One click installer
~~~~~~~~~~~~~~~~~~~

`One click installer`__ is an `AutoIT`__ script that installs Robot
Framework and its dependencies.

__ oneclickinstaller.html
__ http://www.autoitscript.com/autoit3/


Tool for cleaning up Robot Framework test data
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`Robotidy`__ is a tool for cleaning Robot Framework test data, changing test 
data format between HTML and TSV and fixing inline comments to format supported
by Robot IDE.

__ robotidy.html


Tool for diffing Robot Framework outputs
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`Robotdiff`__ is a tool for generating diff reports from Robot
Framework output files.

__ robotdiff.html


Tool for further verification of test results
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`Statuschecker`__ is a tool for verifying that test case statuses and messags 
and also keyword log messages are as expected.

__ statuschecker.html


Test data editing tool (Robot IDE)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Robot IDE is a standalone tool for editing test data. It helps in
creating, editing and maintaining of Robot Framework test data.

More information will be added as soon as Robot IDE project is
made publicly available.
