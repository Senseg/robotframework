   - setting  settings in setting table (e.g. Force Tags), tc/kw tables 
              (e.g [Documentation]) and command line options (e.g. --name)
   - prog     program names (e.g. rebot, risto.py)
   - code     programming code
   - msg      test case status and message, as well as log messages and levels
   - name     keyword, test case and test suite names
   - kw       TODO: rename to 'name' where used
   - cli      command line examples (note that options alone use setting)
   - path     file names and paths
   - misc     everything else (synomyn to ``text``)
