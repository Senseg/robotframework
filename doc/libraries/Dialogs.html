Dialogs - Documentation

Introduction

Dialogs is a test library that provides means for pausing the test execution and asking for user input

Shortcuts

Execute Manual Step  Get Value From User  Pause Execution 

Keywords

Keyword Arguments Documentation
Execute Manual Step message=Set keyword status, error=Execution failed Pauses the test execution and shows a dialog with text message.

User can choose to pass or fail the test. In case of failing the test, additional dialog is opened for defining the error message. error is the default error message.
Get Value From User message=Give value:, default= Pauses the test execution and asks user to input value.

message is the instruction to user. default is the default value of the input field.
Pause Execution message=Test execution paused. Press OK to continue. Pauses the test execution and shows dialog with text message.
